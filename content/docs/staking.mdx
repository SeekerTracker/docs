# Staking

Earn rewards by staking your $TRACKER tokens.

## Overview

Staking rewards receive **10% of all $TRACKER trading volume** revenue. This is distributed to users who stake their tokens.

## How to Stake

1. Hold $TRACKER tokens in your wallet
2. Visit the staking interface
3. Connect your Solana wallet
4. Stake your tokens

## Rewards

- **Source:** 10% of protocol revenue
- **Distribution:** Proportional to stake amount
- **Frequency:** Ongoing accumulation

## API Access

Get staking data programmatically:

```bash
# Vault data
curl https://seekertracker.com/api/skr/vault

# Staker count
curl https://seekertracker.com/api/skr/stakers

# Summary stats
curl https://seekertracker.com/api/skr/summary
```
