# Getting Started with SeekerTracker

Learn how to search, track, and discover .skr SeekerIDs on Solana.

## Searching SeekerIDs

1. Visit [seekertracker.com](https://seekertracker.com)
2. Use the search bar to find any .skr domain
3. View the profile, rank, and registration date
